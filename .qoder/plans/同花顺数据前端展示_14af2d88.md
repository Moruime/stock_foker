# 同花顺原始数据前端展示

## 现状

后端 Agent 已接入 13 个同花顺 Skill，但原始数据仅作为 LLM Prompt 输入，前端只展示 LLM 分析结论。用户无法直接看到行业 PE/PB、资金流向、研报列表、公告原文等数据。

## 设计思路

在每个 Agent 的 `parse_response` 中将关键原始数据附加到返回结果中，前端在现有 LLM 分析卡片下方增加"数据详情"区域展示。采用 Ant Design Collapse/Tabs 组件，默认折叠避免页面过长。

---

## Task 1: 后端 — Agent 结果透传原始数据

修改 4 个 Agent 的 `parse_response()`，将 `raw_data` 中的同花顺原始数据附加到返回字典中。

**文件改动：**

- `backend/app/agents/sentiment_agent.py` — `parse_response` 增加：
  - `hithink_news_raw`: raw_data 中的 hithink_news（前 10 条）
  - `announcements_raw`: raw_data 中的 announcements（前 10 条）

- `backend/app/agents/sector_agent.py` — `parse_response` 增加：
  - `industry_valuation_raw`: raw_data 中的 industry_valuation
  - `market_data_raw`: raw_data 中的 market_data

- `backend/app/agents/macro_agent.py` — `parse_response` 增加：
  - `hithink_index_raw`: raw_data 中的 hithink_index

- `backend/app/agents/enhanced_advice_agent.py` — `parse_response` 增加：
  - `reports_raw`: raw_data 中的 reports_data（前 5 条）
  - `business_raw`: raw_data 中的 business_data
  - `basicinfo_raw`: raw_data 中的 basicinfo_data
  - `shareholders_raw`: raw_data 中的 shareholders_data

**原则**：字段名以 `_raw` 结尾，区分 LLM 分析字段和原始数据字段。

## Task 2: 前端 — SentimentPage 新增新闻/公告数据面板

在 `frontend/src/pages/SentimentPage.tsx` 现有 "重点新闻" 卡片下方新增：

- **同花顺财经资讯** 折叠面板：展示 `hithink_news_raw.data` 列表，每条显示标题、来源、时间、摘要（可点击展开）
- **公司公告** 折叠面板：展示 `announcements_raw.data` 列表，每条显示标题、日期、公告类型

使用 Ant Design `Collapse` 组件，默认折叠。数据为空时不显示。

## Task 3: 前端 — SectorPage 新增行业估值/资金流向面板

在 `frontend/src/pages/SectorPage.tsx` "分析摘要" 卡片下方新增：

- **行业估值指标** 卡片：从 `industry_valuation_raw.datas[0]` 提取 PE/PB/ROE 等数值，使用 Descriptions 或 Statistic 组件展示
- **资金流向** 卡片：从 `market_data_raw.datas[0]` 提取主力净流入、大单净量、换手率、量比，使用 Statistic 组件展示，红绿色区分正负

## Task 4: 前端 — MacroPage 新增指数行情面板

在 `frontend/src/pages/MacroPage.tsx` "关键指标" 卡片上方新增：

- **主要指数行情** 卡片：从 `hithink_index_raw.datas` 提取上证/沪深300/创业板指的收盘价和涨跌幅，使用 Statistic + Row/Col 布局，涨跌用红绿色

## Task 5: 前端 — AnalysisPage 新增数据详情区域

在 `frontend/src/pages/AnalysisPage.tsx` AI 综合分析卡片下方（SnapshotPanel 上方）新增：

- 使用 `Tabs` 组件分 4 个 tab：
  - **研报观点**：展示 `reports_raw.data` 前 5 条，标题、评级、目标价、来源、日期
  - **公司概况**：展示 `basicinfo_raw.datas[0]` 基本信息（行业/上市日期/总股本/总市值）+ `business_raw.datas` 主营构成表格
  - **股东信息**：展示 `shareholders_raw.datas` 前十大股东/实控人，使用 Table 组件
  - tab 仅在对应数据非空时显示
