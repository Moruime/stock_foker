# 同花顺问财 API 查询词修复记录

> 修复日期：2026-04-14

## 1 修复概述

对 `backend/app/services/data_fetcher.py` 中 7 个函数的
查询词进行优化，解决因问财 API 不支持复合查询导致的
数据缺失问题。修复后全部 21 个数据获取函数回归测试通过。

## 2 修复明细

### 2.1 概念板块查询（-2058 错误）

- **函数**：`fetch_concept_boards`
- **原查询**：`{stock_name}所属概念板块 涨跌幅 成份股数量`
- **新查询**：`{stock_name}所属概念板块`
- **原因**：问财 API 不支持概念+涨跌幅+成份股数量的
  复合查询，返回 status_code=-2058
- **影响**：概念板块涨跌幅和成份股数量无法在单次查询中
  获取，但概念名称列表可正常返回

### 2.2 行业板块查询（0 条结果）

- **函数**：`fetch_industry_board`
- **原查询**：`{stock_name}所属行业 行业涨跌幅 行业换手率`
  `行业成交额 行业内涨跌幅前5个股`
- **新查询**：`{stock_name}所属同花顺行业`
- **原因**：原查询条件过于复杂，API 无法解析导致 0 条返回
- **影响**：行业换手率/成交额/前5个股需通过其他拆分查询
  获取（`fetch_hithink_industry_peers` 已覆盖同行数据）

### 2.3 事件数据查询（无事件详情）

- **函数**：`fetch_hithink_events`
- **原查询**：`{stock_name}近期业绩预告解禁减持增持`
- **新查询**：`{stock_name}最新业绩预告 预告类型 预告净利润`
  `变动原因`
- **原因**：复合事件查询仅返回基础股票信息，无事件字段；
  聚焦业绩预告可获取预告类型、净利润中值、摘要、变动原因
- **返回字段**：`变动类型`、`预告净利润中值`、`报告期`、
  `公告日期`、`摘要`、`变动原因`

### 2.4 北向资金查询（返回个股而非汇总）

- **函数**：`fetch_north_flow`
- **原查询**：`最近一个交易日北向资金净流入沪股通深股通`
- **新查询**：`今日北向资金净买入额前10 净买入额 涨跌幅`
- **原因**：问财 API 不提供北向资金汇总数据，始终返回
  个股维度的净买入/净卖出
- **策略**：改为获取北向资金净买入额 Top10 个股，
  为 Agent 提供北向资金流向参考

### 2.5 宏观指标查询（缺 PPI 和 PMI）

- **函数**：`fetch_hithink_macro_indicators`
- **原查询**：
  - `最近一期CPI同比增速 PPI同比增速 PMI数据`
  - `最新LPR利率 M2同比增速 社融数据`
- **新查询**（拆分为 4 个独立查询）：
  - `最近一期CPI同比增速`
  - `最近一期PPI同比增速`
  - `最近一期制造业PMI`
  - `最新LPR利率 M2同比增速 社融数据`
- **原因**：CPI+PPI+PMI 合并查询时 API 仅返回 CPI，
  拆分后各指标均可独立获取
- **返回 key 变更**：`price_activity` 拆分为
  `cpi`、`ppi`、`pmi`

### 2.6 市场涨跌概况查询（数据不精确）

- **函数**：`fetch_market_overview`
- **原查询**：`今日A股行业板块涨跌家数 平均涨跌幅`
  `涨幅前5板块`
- **新查询**：`今日A股上涨家数 下跌家数 涨停家数 跌停家数`
- **原因**：原查询返回的数据粒度不一致，新查询可直接
  获取上涨/下跌/涨停/跌停四项核心指标
